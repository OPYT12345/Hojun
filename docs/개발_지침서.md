## 2. 환경 설정

### 필수 환경변수

| 변수명 | 설명 | 기본값 |
|--------|------|--------|
| `MYSQLHOST` | MySQL 호스트 | `localhost` |
| `MYSQLPORT` | MySQL 포트 | `3306` |
| `MYSQLUSER` | DB 사용자명 | `root` |
| `MYSQLPASSWORD` | DB 비밀번호 | `1234` |
| `MYSQLDATABASE` | DB 이름 | `school` |
| `SSL_ENABLED` | HTTPS 활성화 여부 | `true` (Railway에서는 `false`) |
| `SSL_PASSWORD` | keystore.p12 비밀번호 | `school2026` |
| `COOKIE_SECURE` | Secure 쿠키 여부 | `true` (Railway에서는 `false`) |
| `OPENAI_API_KEY` | OpenAI API 키 | (없으면 AI 기능 비활성) |

### application.properties 주요 설정

```properties
# HTTPS
server.port=8443
server.ssl.enabled=${SSL_ENABLED:true}
server.ssl.key-store=classpath:keystore.p12

# JPA
spring.jpa.hibernate.ddl-auto=update   # 테이블 자동 생성/수정
spring.sql.init.mode=always            # data.sql 매 실행 시 반영

# 파일 업로드
spring.servlet.multipart.max-file-size=50MB

# AI 예산 한도
app.ai.max-calls-per-user-per-hour=20
app.ai.max-calls-global-per-day=300
```


### Railway 클라우드 (라이브 배포)
Railway 대시보드에서 다음 환경변수를 설정한다:

|--------|-----|

| `SSL_ENABLED` | `false` (Railway가 HTTPS 처리) |
| `COOKIE_SECURE` | `false` |
| `OPENAI_API_KEY` | 발급받은 키 입력 |

## 4. API 설계 원칙

### 인증
- 모든 API는 세션(`userId`) 유무로 인증 확인
- 미인증 시 `401` 반환
- 권한 부족 시 `403` 반환
- 잘못된 입력값 `400` 반환

### URL 구조
```
/api/student/*          학생 관련 (로그인, 회원가입, 상태)
/api/teacher/*          강사 관련 (로그인, 회원가입, 강의 목록)
/api/lecture/{id}/*     강의별 리소스 (자료, 과제, Q&A, 공지, 출석)
/api/notifications/*    알림
/api/ai/*               AI 기능
/api/user/*             사용자 공통 (비밀번호 변경 등)
/nfc/{lectureId}        NFC 단축 URL
```

### 강의 소유권 검증
강사가 강의별 리소스를 수정/삭제할 때는 반드시 `lectureRepo.existsByIdAndTeacherId(lectureId, userId)` 검증 수행.

## 5. 프론트엔드 규칙

### sessionStorage 키
| 키 | 설명 |
|----|------|
| `lecId` | 현재 입장한 강의 ID |
| `lecName` | 현재 강의 이름 (헤더 표시용) |

### localStorage 키
| 키 | 설명 |
|----|------|
| `studentName` | 로그인한 학생 이름 |
| `teacherName` | 로그인한 강사 이름 |
| `theme` | 라이트/다크 테마 설정 (`light` / `dark`) |

### XSS 방지
모든 서버 데이터를 HTML에 렌더링할 때 반드시 `esc()` 함수 사용:
```javascript
function esc(s) {
  return String(s||'').replace(/&/g,'&amp;').replace(/</g,'&lt;')
                      .replace(/>/g,'&gt;').replace(/"/g,'&quot;');
}
```

### 테마 시스템
전체 13개 HTML 파일에 라이트/다크 테마 적용. `<body>` 직후 early 스크립트로 FOUC(깜빡임) 없이 즉시 테마 적용.
```javascript
const t = localStorage.getItem('theme') || 'dark';
if (t === 'light') document.body.classList.add('light');
```
